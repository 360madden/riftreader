# Historical recovery artifacts

This folder stores superseded recovery truth and proof-anchor artifacts for audit/history only.

**Do not use files in this folder as current live truth.** Current operating truth is:

- `docs/recovery/current-truth.md` — concise human dashboard.
- `docs/recovery/current-truth.json` — machine-readable canonical truth.

Historical files may contain old PIDs, HWNDs, absolute addresses, expired RRAPICOORD references, or stale proof-anchor caches.
